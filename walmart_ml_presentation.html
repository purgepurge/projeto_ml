Resumo do Projeto

Desenvolvimento de modelo preditivo para estimar vendas semanais utilizando técnicas de Machine Learning supervisionado. Dataset com 6.435 registros de 45 lojas diferentes (2010-2012).

Slide 1 de 10

Contexto e Problema

Desafios do Varejo Moderno

  • Sazonalidade e tendências temporais
  • Variabilidade entre diferentes unidades
  • Influência de fatores econômicos externos
  • Impacto de eventos especiais e feriados

Por que Machine Learning?

  • Captura relações não-lineares
  • Processa múltiplas variáveis simultaneamente
  • Superior aos métodos estatísticos tradicionais
  • Adaptável a novos padrões

Impacto: Previsões imprecisas geram custos significativos por excesso de estoque ou vendas perdidas. Melhorias na acurácia podem reduzir custos em até 30%.

Slide 2 de 10

Objetivos do Projeto

Objetivo Geral

Desenvolver um modelo de Machine Learning capaz de prever vendas semanais com alta precisão, utilizando algoritmos clássicos de aprendizado supervisionado.

Objetivos Específicos

  • Realizar análise exploratória detalhada do dataset
  • Implementar pipeline completo de pré-processamento
  • Treinar e comparar três algoritmos: Linear, Ridge e Random Forest
  • Avaliar desempenho com métricas apropriadas (RMSE, MAE, R²)
  • Aplicar validação cruzada e controle de overfitting
  • Identificar variáveis mais relevantes para predição
  • Fornecer recomendações práticas baseadas nos resultados

Principais Insights

Heterogeneidade entre Lojas

68,3% da importância vem das características individuais das lojas. Cada unidade tem padrões únicos relacionados a:

  • Localização geográfica
  • Tamanho da loja
  • Demografia local
  • Concorrência regional

Impacto Econômico

Desemprego e CPI são preditores significativos (18,7%), confirmando que o cenário macroeconômico afeta o poder de compra dos consumidores.

Padrões Sazonais

Variáveis temporais (Month, Week) contribuem 9,4%, refletindo comportamento sazonal:

  • Períodos de férias
  • Back-to-school
  • Festas de fim de ano

Relações Não-Lineares

A superioridade do Random Forest (16,2% melhor) evidencia que interações complexas entre variáveis são cruciais para predições precisas.

Conclusões e Aplicações

Resultados Alcançados

  • Modelo com 94,78% de precisão (R²)
  • Erro médio de apenas 6,5% das vendas
  • Identificação dos principais drivers
  • Pipeline completo e reprodutível

Aplicações Práticas

  • Planejamento de estoque otimizado
  • Alocação eficiente de recursos
  • Decisões estratégicas baseadas em dados
  • Identificação de lojas atípicas

Limitações

  • Período temporal limitado (3 anos)
  • Ausência de dados sobre promoções
  • Sem informações de concorrência
  • Agregação semanal (não diária)

Próximos Passos

  • Adicionar features de promoções

  • Implementar em ambiente de produção

Mensagem Final

O projeto demonstra com sucesso a aplicação de Machine Learning para resolver problemas reais de negócio, oferecendo uma ferramenta viável e precisa para previsão de vendas no varejo.
